# React + Vite Project

## 🚀 설치 및 실행 방법

### 1. 프로젝트 생성
- **현재 경로에 프로젝트 구성**
  ```bash
  npm create vite@latest .
  ```
  
- **새 폴더 생성 후 구성**
  ```bash
  npm create vite@latest <project-name>
  ```

### 2. 프로젝트 이동
 ```
bash
cd <project-name>
 ```
### 3. 의존성 패키지 설치
 ```
bash
npm install
 ```
### 4. 개발 서버 실행
```
bash
npm run dev
```
